Who and why?

Agencies & freelancers

No more vague emails or improvised screenshots: clients comment directly on the page, you export everything in one clean file.

Product & UX teams

Review where the interfaces live: in the browser. Uxnote adds notes and highlights without touching existing code.

Core features

Capture modes

Text, element, free area with priorities, collapsible snippets, and numbered badges.

Site-wide

Global list for the whole domain, jump-to opens the page and centers the annotation.

Import / Export / Mail

Single JSON (title + date), import with full re-render, email sending for sharing with devs.

Install (copy / paste)

Drop Uxnote right before </body> on each page you want to annotate:

<script src="https://uxnote.ninefortyone.studio/uxnote-tool/uxnote.js"></script>
1

Inject

Paste the snippet on each page (or via a global tag manager).

2

Share

Send the URL to your client (optionally add ?uxnote=1 to flag review mode).

3

Annotate

Clients highlight, comment on text/element/area; everything shows in the Uxnote panel.

4

Export / Mail

Dev side: click “Export” (JSON) or “Mail” to collect and process feedback.
