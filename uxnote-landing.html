Pour qui et pourquoi ?

Agences & freelances

Finis les mails vagues et les captures bricolées : tes clients commentent directement sur la page, tu exportes tout en un seul fichier lisible.

Équipes produit & UX

Faites vos revues là où les interfaces vivent : dans le navigateur. Uxnote ajoute une couche de notes et surlignages sans toucher au code existant.

Fonctionnalités essentielles

Modes de capture

Texte, élément, zone libre avec priorités, snippets repliables et badges numérotés.

Site-wide

Liste globale pour tout le domaine, jump-to : un clic ouvre la page et centre l'annotation.

Import / Export / Mail

JSON unique (titre + date), import avec re-render complet, envoi par email pour partager les retours côté dev.

Installation (copier-coller)

Ajoute Uxnote juste avant </body> sur chaque page à annoter (ton script est sur GitHub) :

<!-- Uxnote --> <script src="https://cdn.example.com/uxnote.js" data-icons="/ressources" data-position="bottom"></script>

Charge Uxnote sur toutes les pages où tes clients commentent. La liste regroupe tout le site et le click sur une annotation ouvre la page cible puis focus l’élément.

1

Injecter

Colle le snippet sur chaque page (ou via un tag manager global).

2

Partager

Envoie l’URL au client (tu peux ajouter ?uxnote=1 si besoin d’activer un mode dédié).

3

Annoter

Le client surligne, commente texte/élément/zone; tout apparaît dans le panneau Uxnote.

4

Exporter / Mail

Côté dev, clique “Exporter” (JSON) ou “Mail” pour récupérer et traiter les retours.

Comment l’utiliser avec vos clients

1

Prépare

Maquette ou préprod comme d’habitude, Uxnote déjà inclus dans le template.

2

Partage

Envoie à ton client l’URL (optionnel : ?uxnote=1 pour marquer les revues).

3

Annoter

Le client annote, commente, surligne sans installer quoi que ce soit.

4

Récupérer

Tu exportes les annotations en JSON ou via email pour les intégrer dans ton workflow.
